## Usage

CeStyle does not expose typography global styles.
Use TypographyStylesProvider to add CeStyle typography styles to your html content:

```tsx
import { TypographyStylesProvider } from '@cestyle/core';

function Demo() {
  return (
    <TypographyStylesProvider>
      <div dangerouslySetInnerHTML={{ __html: '<p>Your html here</p>' }} />
    </TypographyStylesProvider>
  );
}
```
